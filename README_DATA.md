# Data directory

No patient-level data, raw CTG traces, synthetic datasets, or manuscript result CSVs are included.

This directory contains schema-only CSV files:

- `analysis_input_schema.csv`: expected individual-level analysis-dataset columns;
- `contrast_estimates_schema.csv`: expected contrast-level columns for E-values and multiplicity labels;
- `cohort_estimates_schema.csv`: expected cohort-level columns for random-effects pooling.
